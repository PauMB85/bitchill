# BitChill - front

## Instalacion

```bash
npm install
```

## Arrancar proyecto

crear fichero .env e incluir

```bash
VITE_API_ADDRESS_SM_HELLO_WORLD=ADDRESS_SMART_CONTRACT_HELLO_WORLD
VITE_API_NODE_ENDPOINT=URL_NODE_LOCAL
VITE_API_BACKEND_ENDPOINT=ULR_BACKEND

VITE_CLIENT_ID=ID_WEB3_AUTH
```

```bash
npm run dev
```
